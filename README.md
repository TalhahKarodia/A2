# Intelligent Submission and Review System

This project contains two separate Java implementations of the system:
- **baseline/**: The original, unoptimized system (as per the initial sequence diagram)
- **optimized/**: The refactored, optimized system (with improved design and logic)

## How to Find Each System
- The `baseline` folder contains `BaselineSystem.java` for the baseline implementation.
- The `optimized` folder contains `BaselineSystem.java` for the optimized implementation.

## How to Run Each System

### 1. Open a Terminal
- On Windows, you can use Command Prompt, PowerShell, or WSL.

### 2. Navigate to the Desired Folder
- For the baseline system:
  ```
  cd "A2/baseline"
  ```
- For the optimized system:
  ```
  cd "A2/optimized"
  ```

### 3. Compile the Java File
```
javac BaselineSystem.java
```

### 4. Run the System
```
java BaselineSystem
```

### 5. Observe the Output
- The system will print the process flow and results to the console.

---

**Tip:**
- Make sure you are in the correct folder before compiling and running.
- You can compare the outputs of both systems for analysis and reporting.
